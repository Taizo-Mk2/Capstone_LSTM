# Capstone_LSTM
大学のCapstone Projectとして「回帰性ニューラルネットワークの1つであるLSTM（長・短期記憶）を使ったオンラインゲームのアクティブユーザー数の予測モデル構築」に取り組みました。 
直近30日のデータからデータ取得日から7日分のアクティブユーザー数を予測するモデルです。

## 概要
オンラインゲームの販売プラットフォームの一つであるSteamのデータを用いて、予測モデルを構築しました。
学習データには、日毎のアクティブユーザー数、ゲーム価格、アップデートやダウンロードコンテンツ（DLC）のリリースなどのイベント情報を使用しました。
時系列データのパターンを捉えるため、回帰型ニューラルネットワーク（Recurrent Neural Network, RNN）の一つであるLSTM（長・短期記憶、Long Short-Term Memory）を使用しました。

## LSTMとは？
回帰型ニューラルネットワーク（RNN）は、学習の過程で隠れ層から出力された値を次の時刻の学習に取り入れるという特徴を持ちます。
LSTMでは、その値を取り入れる前に「忘却ゲート」などのゲート構造を通すことで、保持する情報と捨てる情報を調整します。これにより、long short-term memory(長期的な依存関係)を学習しやすくなります。


## 使用データセット
Steamのデータを収集している『SteamCharts』から、５つのジャンルのデータを収集しました。
対象ジャンルはAction、FPS、Free to Play、Grand Strategy、Indieの５つです。
各ジャンルにおいて、ピーク時のアクティブユーザー数が上位１０位までのゲームのデータを使用し、合計50本のゲームを対象に予測モデルを構築しました。
学習の際にはそれぞれの値を正規化しました。こちらのデータは「Normalized Data」フォルダにまとめています。
Githubには「Elden Ring」と「Terraria」のデータをアップロードしています。


## 目的変数・独立変数
目的変数は、データ取得日から7日分のアクティブユーザー数です。
独立変数は、日毎のアクティブユーザー数、ゲーム価格、アップデートやダウンロードコンテンツ（DLC）のリリースなどのイベント情報です。
ゲーム価格はセールや無料化などにより日々変動するため、価格についても日毎のデータを使用しました。



## セッティング
- 入力データ：３０日分のアクティブユーザー数データ
- 出力データ：７日分のアクティブユーザー数データ
- 特徴量数：３（アクティブユーザー数、価格、イベント情報）
- バッチサイズ：32
- エポック数（学習回数）：100
- LSTMのユニット数：64
- アクティベーション関数：tanh (hyperbolic tangent、双曲線関数) 
- ドロップアウト率：0.2
- 検証データの割合：0.1
- テスト（検証）期間：90日分のデータ

## アップロードしているもの
Actionから「Elden Ring」、「Terraria」のコードをアップロードしています。Terrariaは大規模アップデート直後にアクティブユーザー数の跳ね上がりがあり、上記のセッティングでは跳ね上がりを予測できていなかったため、イベント情報を大規模アップデートと小規模アップデートの２種類に分け、特徴量数を4、学習回数を200に設定しました。
